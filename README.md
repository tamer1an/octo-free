# Github File Tree (Octofree)

A browser extension that adds a navigable file tree sidebar to GitHub repositories, with support for private repos, branch switching, fast file search, and direct file downloads.

> Originally forked from [Octotree v2](https://github.com/ovity/octotree) by Buu Nguyen and contributors.  
> Rebuilt from scratch using **React 18**, **TypeScript**, **Vite**, and **Manifest V3**.

![Artboard 1](https://user-images.githubusercontent.com/956869/60222787-ec57f780-987e-11e9-9b71-e4db49dabf9b.png)

## Features

- 🌲 **File tree sidebar** — Browse any GitHub repository's file structure at a glance
- 🔀 **Branch switching** — Switch between branches without leaving the page
- 🔍 **File search** — Instantly find files using the built-in search with autocomplete
- ⬇️ **Direct downloads** — Download individual files with a single click
- 🔒 **Private repo support** — Optionally provide a GitHub Personal Access Token
- 🌙 **Dark "Cartoon Smart Studio" theme** — A distinctive, eye-friendly dark UI

## Installation

Install from the [Chrome Web Store](https://chrome.google.com/webstore) (search for "Github File Tree (Octofree)").
