Octo-Free adds a collapsible file tree sidebar to any GitHub repository page, 
making it easy to navigate large codebases without leaving the page.

Features:
• File tree sidebar — browse the full repository structure at a glance
• Branch switcher — switch between any branch directly from the sidebar
• File search — quickly locate files by name using autocomplete
• File download — download any individual file with one click on file ico
• GitHub token support — set a Personal Access Token in Options to avoid 
  API rate limits and access private repositories

Supercharge your GitHub experience with a lightning-fast code tree sidebar. Navigate, search, and download files instantly! 🐙

Detailed Description
Navigate GitHub like a Pro! Octo-Free transforms how you explore repositories on GitHub. Stop clicking through endless folders and waiting for pages to load. Octo-Free injects a beautifully designed, highly interactive file tree directly into your browser, allowing you to explore any repository instantly, just like you would in your favorite IDE.

Built from the ground up using modern React and Vite, Octo-Free is incredibly fast, lightweight, and features a stunning "Cartoon Studio" dark mode aesthetic that makes reading code a joy.

![Artboard 1](https://user-images.githubusercontent.com/956869/60222787-ec57f780-987e-11e9-9b71-e4db49dabf9b.png)

✨ Key Features:

🌲 Lightning-Fast Tree Navigation: Explore entire repository structures instantly. Navigate nested folders without ever reloading the page.
🔍 Instant File Search: Looking for a specific file? Use the built-in, ultra-fast autocomplete search bar to jump to any file in milliseconds.
🔀 Seamless Branch Switching: Switch between branches or tags directly from the sidebar. The tree updates automatically.
📥 One-Click Raw Downloads: No need to open a file just to save it! Download any raw file directly from the tree view with a single click.
🎨 Stunning "Cartoon Studio" UI: Enjoy a premium, beautifully crafted dark-mode interface featuring vibrant accents, smooth micro-animations, and easy-to-read typography (powered by the Nunito font).
🚀 Reload-Free Browsing: Octo-Free seamlessly hooks into GitHub's native routing. Clicking a file instantly displays it in GitHub's viewer without a hard page refresh.
🔒 Private Repo Support: Securely enter your GitHub Personal Access Token in the extension options to browse your private repositories and completely bypass public GitHub API rate limits.
🛡️ Privacy First: Octo-Free respects your privacy. We do not track your browsing history or collect personal data. The extension only activates on github.com. Your optional GitHub Personal Access Token is encrypted and securely stored locally using Chrome's native sync storage, meaning it never touches a third-party server.

Say goodbye to endless clicking and take control of your GitHub workflow today with Octo-Free!
